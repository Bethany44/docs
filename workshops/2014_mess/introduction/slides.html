What does it do?

Provide complete framework to jointly work with any type of seismological data in Python

  • unified object classes
  • read/write file format support
    • waveforms (MiniSEED, SAC, GSE, SEISAN, SEG-Y, ...)
    • metadata (StationXML, SEED, QuakeML, ...)
      •
  • data center access
    • FDSN web services (IRIS, ...), ArcLink (EIDA, ...), Neries/Nera, Earthworm, Seishub, SeedLink, ...
      •
  • basic signal processing routines
    • filter, instrument correction, trim, merge, taper, rotate, resample, ...
    • array analysis, CC routines, PPSD, triggers, cpx trace analysis, ...
      •

What's the beauty of it?

  • Runs on all Platforms
    • Windows, Mac, Linux, FreeBSD (Installers, Debian/Ubuntu repository, ...)
    • from RaspberryPi (35$) to SuperMUC (>150 Mio.$)
  • Free and open source (LGPL)
  • Community pages: GitHub, wiki, tutorials, gallery, installation instructions, ticket system, mailing list
  • Well documented source code, auto-generated API docs
  • Test-driven development (1100+ unit tests, 82% Test Coverage, CI)
  • Reuses well established third-party code
    • Python community: numpy, scipy, matplotlib, ...
    • Seismo community: libmseed, GSE UTI, evalresp, ...
      •
  • Python!

Python!

  • Quick to learn, very concise and easy-to-read syntax
  • One of the most used programming languages
  • Free and Open Source, large community
  • Cross-platform: Runs everywhere from RaspberryPi to large supercomputers
  • No need to compile, interactive shell available
  • General purpose language (in contrast to many other tools used in science)
  • "Batteries included" (debugging, http access, ...)
  • Mature third party libraries (plotting, big data, ..)
  • Easy to interact with existing C and Fortran code

Do people actually use it?

  • ~270 people on users mailing list
  • 40 scientific citations (thanks guys!)

Who's behind it?

  • development started in 2008
  • ~4 core developers in Munich
  • since then around 30 people from all around the world contributed
  • 5000+ code commits
  • 700+ tickets and pull requests
